## Youtube Clone

Bu proje, YouTube'un temel işlevlerinden bazılarını, örneğin videoları izleme, video detaylarını görüntüleme ve yorumları okuma işlevlerini sunmaktadır.
 
## Kullanılan Teknolojiler

-HTML
-CSS
-JavaScript
-React
-API
